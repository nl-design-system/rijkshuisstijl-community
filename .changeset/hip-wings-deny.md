Add class names to enable fluid design and dark mode:

- `rhc-theme--fluid`: fluid design tokens for all the things
- `rhc-theme--fluid-text`: fluid typography tokens
- `rhc-theme--fluid-space`: fluid space tokens for all the things
- `rhc-theme--dark-mode`: enable switching to dark mode

The modifier class names are available for the other theme class names too.
