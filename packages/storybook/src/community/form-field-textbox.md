# Rijkshuisstijl Community Form field with text input

## Usage

```jsx
import { FormFieldTextbox } from '@rijkshuisstijl/community-components';

<FormFieldTextbox
  label="Label"
  placeholder="Placeholder"
  description="Description"
  value="Value"
  onChange={onChange}
  error="Error"
  required
  disabled
  readOnly
/>;
```
