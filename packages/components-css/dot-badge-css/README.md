# Omschrijving

Dit package bevat de specifieke code voor het component. Als je alle rijkshuisstijl community componenten wil kijk dan naar [https://github.com/nl-design-system/rijkshuisstijl-community/blob/main/packages/components-css/README.md]. Voor verdere informatie over het inzet van de component kijk naar onze documentatie [https://rijkshuisstijl-community.vercel.app/?path=/docs/css-dot-badge--docs]

## Installatie

```bash
pnpm add @rijkshuisstijl-community/dot-badge-css
```

## Gebruik

```html
import '@rijkshuisstijl-community/dot-badge-css';

<span class="rhc-dot-badge" role="status">
  <span class="rhc-dot-badge__sr-only" data-testid="rhc-dot-badge__sr-label"> Nieuw bericht </span>
  <svg />
</span>
```
