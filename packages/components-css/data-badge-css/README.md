# Omschrijving

Dit package bevat de specifieke code voor het component. Als je alle rijkshuisstijl community componenten wil kijk dan naar [https://github.com/nl-design-system/rijkshuisstijl-community/blob/main/packages/components-css/README.md]. Voor verdere informatie over het inzet van de component kijk naar onze documentatie [https://rijkshuisstijl-community.vercel.app/?path=/docs/css-data-badge--docs]

## Installatie

```bash
pnpm add @rijkshuisstijl-community/data-badge-css
```

## Gebruik

```html
import '@rijkshuisstijl-community/data-badge-css';

<span class="rhc-data-badge" role="status"> tekst </span>
```
