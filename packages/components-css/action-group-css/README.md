# Omschrijving

Dit package bevat de specifieke code voor het component. Als je alle rijkshuisstijl community componenten wil kijk dan naar [https://github.com/nl-design-system/rijkshuisstijl-community/blob/main/packages/components-css/README.md]. Voor verdere informatie over het inzet van de component kijk naar onze documentatie [https://rijkshuisstijl-community.vercel.app/?path=/docs/css-action-group--docs]

## Installatie

```bash
pnpm add @rijkshuisstijl-community/action-group-css
```

## Gebruik

```html
import '@rijkshuisstijl-community/action-group-css';

<p role="group" class="utrecht-action-group utrecht-action-group--column">
  <button class="utrecht-button utrecht-button--primary-action" type="button">
    Save and continue
  </button>
  <button class="utrecht-button utrecht-button--secondary-action" type="button">
    Back
  </button>
</p>

```
